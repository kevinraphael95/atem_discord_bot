📋 Général 4 commandes ›
!code
Affiche un lien vers le code source du bot.
!help
Affiche l'aide du bot.
!ping
Affiche la latence actuelle du bot.
!say
Fait répéter un message par le bot. Options : *embed / *e, *as_me / *am.
!say *e *am Bonjour !

Affiche les cartes d'une banlist (tcg, ocg ou goat) avec pagination.

🔍 Rechercher une carte ou tirer aléatoirement avec !carte random.

Affiche le prix d'une carte Yu-Gi-Oh!
!sets
📦 Affiche tous les sets d'une carte avec rareté, prix et date TCG.
!staples
Affiche des cartes considérées comme des staples.
!vocabulaire
📘 Affiche les définitions des termes Yu-Gi-Oh! avec navigation interactive.
!ygotuto
Tutoriel interactif pour apprendre à jouer à Yu-Gi-Oh!
