Installation

Tout ce qu'il faut pour faire tourner Atem sur ton propre serveur Discord.

Étapes d'installation

01

Prérequis

Avant de commencer, assure-toi d'avoir :

Python 3.10+ pip Git Compte Discord Dev

Un bot Discord créé sur le Discord Developer Portal avec les intentions Message Content et Server Members activées.

02

Cloner le dépôt

Récupère le code source depuis GitHub :

git clone https://github.com/kevinraphael95/atem.git cd atem
03

Installer les dépendances

Installe toutes les bibliothèques Python requises :

pip install -r requirements.txt
Conseil : Utilise un environnement virtuel Python (python -m venv venv) pour isoler les dépendances.
04

Configurer le token

Crée un fichier .env à la racine du projet :

# .env DISCORD_TOKEN=ton_token_ici
Important : Ne partage jamais ton token. Ajoute .env à ton .gitignore.
05

Lancer le bot

Démarre Atem avec :

python bot.py

Si tout est correct, le bot apparaîtra en ligne. Teste avec !help.

06

Synchroniser les commandes slash

Pour activer les commandes slash Discord, lance dans un salon :

!sync

Les commandes slash seront disponibles après quelques secondes.
