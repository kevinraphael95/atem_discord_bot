Installation du Bot

1️⃣ Prérequis

  • Python 3.10+
  • Bot créé sur Discord Developer Portal

2️⃣ Installer les dépendances

pip install -r requirements.txt

3️⃣ Lancer le bot

python bot.py

4️⃣ Configuration

Ajoutez votre token Discord dans un fichier config ou variable d’environnement.
